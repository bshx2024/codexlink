# 🛠️ Codex 知识库控制中心 (Control Center)

此目录存放了您与 Codex 智能代理协同所需的提示词模板与脚本规则。当您需要处理知识库时，直接复制以下相应的**黄金提示词（Golden Prompts）**在与 Codex 的对话框中发送即可。

---

## 🔄 核心阶段 1：自动化知识蒸馏 (A ➔ B)

当您在 `01_原始资料_Raw` 中积累了大量的原始文章、网页剪藏或剪贴板内容后，您可以向 Codex 发送以下命令：

> 💡 **黄金提示词：**
> 
> "帮我扫描 `01_原始资料_Raw` 文件夹。分析里面所有新进未处理的资料，提炼出其中最有价值的核心概念、逻辑关系和事实。使用结构化的卡片笔记格式，为每个独立概念在 `02_消化复盘_Digest` 中生成一篇深度笔记，并使用双向链接 `[[文件名]]` 将它们关联起来。完成后，在原始文档顶部标记 `[已消化]`。"

---

## 🛠️ 核心阶段 2：方法论沉淀 (B ➔ C)

当您希望将零散的知识转化为可操作的工作流、Checklist 或思考框架时：

> 💡 **黄金提示词：**
> 
> "分析 `02_消化复盘_Digest` 目录下的核心笔记，特别是关于 `[具体主题]` 的内容。为我总结出一套可复用的实操方法论模板，写清楚适用场景、核心步骤和避坑指南，并将其作为模板文件保存到 `03_方法论模板_Templates` 文件夹下。"

---

## ✍️ 核心阶段 3：高效内容输出 (C + B ➔ D)

当您准备产出文章、PPT 大纲、周报或策划案时：

> 💡 **黄金提示词：**
> 
> "我想基于 `03_方法论模板_Templates` 目录下的方法论，结合 `02_消化复盘_Digest` 中的素材，写一篇关于 `[具体话题]` 的深度内容。请调用我保存在 `99_Codex_Skills/style_guide.md` 中的个人写作风格（文风 Skill），生成一份结构完整、逻辑严密的高质量草稿，保存到 `04_最终输出_Output` 目录中。"

---

## 🔁 核心阶段 4：复盘回流闭环 (D ➔ 知识库)

在项目结束或内容发布后，为了让知识库完成自生长闭环：

> 💡 **黄金提示词：**
> 
> "我刚刚完成了 `04_最终输出_Output` 中的 `[项目/文章名称]`。请阅读这份最终产出，提炼出我们在执行过程中学到的新经验、发现的新工具或犯过的错误，将其作为新的知识卡片增补回 `02_消化复盘_Digest` 中，完成自我进化循环。"
